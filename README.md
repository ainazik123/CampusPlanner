Project Structure
All classes are located inside a separate package inside src: Package name: campuslife
Classes Overview
1️⃣ Course
Represents a university course.
2️⃣ AssignmentTask
Represents coursework related to a course.
3️⃣ StudySession
Represents time spent studying for a course.
4️⃣ Main
The driver class that demonstrates:
Creating courses
Creating assignments
Creating study sessions
Printing:
All courses
All assignments (highlighting urgent ones)
Total remaining estimated hours
Total study time per course
Marking one assignment as completed and recalculating totals
